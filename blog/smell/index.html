[体験記]自臭症はほんまに怖い病気だった件

life

友達から自臭症だと打ち明けられた人 「そもそも自臭症ってなんだろう？どう接したほうがあいつのためになるかな？わからないどうしよ、、、、、」

このblogではそういった疑問を解決していきます。

本記事のテーマ

自臭症はどんな症状なのか。どう対応するべきか。

今回の道標

  • ①自臭症 : そもそもどんな症状
  • ②辛さ : 本人の思い
  • ③向き合い方 : 周りにどう対応して欲しいか
  • ④最後に : まとめ・生きる

記事の信頼性

筆者本人が自臭症です。

「ほんとに臭いんだ！おならが勝手にでてる、口臭が向かいのホームの人にまで届いてる、わきがかも、周りに毎回迷惑かけてる、ほんとに生きてる意味あんのかな」

こんな思いが年中頭が勝手に考えてしまうのです。そりゃ頭おかしくなっても不思議じゃないですよね？

実際ほんとに臭いと思われてるかもしれない、それが原因でいじめられてるかもしれない、頭では多分ずっとこの事とは向き合っていく事になると思います。

でも少し楽になれる考え方をこのblogで紹介していきます。まーこんな感じに苦しんでる人の気持ちがわかるので、そこが記事の信頼性なんじゃないかなと思います。

前置きメッセージ

この記事では「自臭症になった筆者の一意見を述べます。」ほんとにきつくて死にたいって人の気持ちもよくわかります。

俺・わたしはもっと苦しんでるって思う人は中にはいると思います。ここのblogでは、批判するのではなく思いを一緒に共感できたらいいなと筆者は思っております。

①自臭症 : そもそもどんな症状

自臭症は自分がほんとに臭いか臭くないのか関係なしで、

自分のにおいのことを年中考えてその度に気持ちが沈むっていうのが僕の思う症状だと思います。

臭い・臭くないは大事だけど大事じゃない

僕自身、臭いに囚われすぎて外にも一歩に出れない日もありました。

電車にのると必ず僕の周りの人が鼻をすするのです。マスクの上から鼻をさすったりしています。

席の後ろの人はむせたり、周りで誰かが「なんか臭くない？」って言っていたり、、、

↑自臭症経験した方分かりますよね？これほんまきついですよね

でも乗り越えるには一番大事なポイントは「実際に臭い・臭くないかを考えすぎない」ことだと思います。自臭症の人は自分が今におっているか死ぬほど心配します。におっていたらデオトラントのぬるタイプのやつを使ったり、手術したりなどいろいろ対策はあります。あまり考えすぎないように努力しましょう！

②辛さ : 本人の思い

辛さでいったら、何にも言い表せないほどきついです。かなりの人がこれを原因として亡くなったりするほどです。

ほんとに生きる意味が見出せなくなります。でも自臭症になった人はメンタルが生きる上でとても大事だという事を周りの人以上に知っています。これはすごい強みになると僕は考えます。

あんな強そうなガタイのいいスポーツ選手だってメンタルコーチを雇ってるんです。スポーツの世界もきついけど、この世の中を生きぬくのも相当なメンタルを使うはずです。そんな中メンタルコーチもつけず、思考錯誤してる僕たちって凄くないですか？

この人並み以上にメンタルが大事だと知ってる僕たちだからこそ積極的に休みを取り入れ、できない事はできないって言える勇気を持つべきだと考えます。

飲み会しんどかったら断ってみる、それで関係をを切られるくらいならこっちからお断りだ！

一人の時間を大切にしよう！運動してみたり、読書してみたり、甘いもの食べたり、ゴロゴロ漫画読んだり、午前中いっぱい寝たり、、、、、

自臭症で性格が歪む？

なんか周りが一気に信用できなくなるんですよね。だからおとなしくなったり、あんま喋んなくなったり、自分を出さなくなりますね。

あと少しきれっぽくなったりしますね。

③向き合い方 : 周りにどう対応して欲しいか

できれば優しく普段通り

自臭症だから気を遣わなきゃと思わないようにしてください。特別な扱いはしないように、笑顔でなるべく接してください。

普段通りが一番です。

接し方の順序

  • ① : 少し暗くなった時
  • ② : 元気がない時
  • ③ : 自臭症発覚した時
  • ④ : 治療している時

① : 少し暗くなった時

そんなに踏み込まずに笑顔を増やして、会話を心がけてください学校の話などを聞き込むのはNG!

② : 元気がない時

なんかあったの？元気ないからあれだったら相談乗るよって、なるべく踏み込まずにきいてください。

③ : 自臭症発覚した時

この場合病院にいって、薬とかもらうのですが、なにがあっても怒らないことが大事です。ほんとに味方であることを伝えることが大事です。

病院(精神科)に行くか・行かないかの基準は周りの目線が気になって集中できない、鼻すすりが気になってしょうがないなどの症状があったらなるべく早く行ったほうがいいですね。相談することで心が楽になる場合もあると思うので。

④ : 治療している時

笑顔がなんだかんだすごい大事ですね。笑顔で優しくなるべく会話をすることを心がけてください。

④最後に : まとめ・生きる

僕たちは100年しか生きられないって言い聞かせてみてください。なんか少し短いかもって感じません？そんな少ない時間で周りの人のこと考えて生きるより自分に時間を使って欲しいです。

もしほんとに、におっていて、いじめられたらリーダー格ぶん殴ればいいです。全員にいじめられたらそこの環境があなたの居場所ではないんです。

学校だったり、職場だったり、サークルだったりあなたが所属している中で人間関係があります。そこから外れてみるという選択肢は全然ありです。自分一人で稼ぐ仕事につくってのもありです。僕の場合blogで稼いでいます。気になった方はblogで稼ぐっていう記事読んでみてね。

まーここでみなさんに、自臭症は怖いってことを鮮明に伝えられたのではないかなと思います。

最後に

自臭症で今苦しいって方へほんとにきついかもしれないけど、少なからず何十億人いる地球にはあなたの仲間は必ずいます。僕はその内の一人です。

お風呂にも入れない国が世界にはあります。アフリカなど発展途上国に旅行などに行くと価値観が変わって全然違う視点で物事を見れるようになるかもしれません。世界一周の旅に出て美しい景色を見ることで自分の悩んでいたことがちっぽけに見えるかもしれません。

お金に余裕があったら遠い異国の地にいってみるのもありですね！

周りの人を見てください。みんな元気そうに見えてもみんな悩みをそれぞれ抱えています。一回限りの人生ですから当然ですが、、

みんな死ぬんです、死んだあとは何も残らない。そう思って、周りを気にせず生きていきましょう！周りはどうとでも思っとけていうスタンスで生きていきましょう！！

人の目を気にせず自分の人生を歩んでいこう

最後まで読んでくださりありがとうございます！
