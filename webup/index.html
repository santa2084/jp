web公開の仕方

こんにちは、ぽてっとです。今回は、webサイトの公開の手順を説明していきます。

下記にロードマップを用意しました。

  • ① : html・cssファイルの準備
  • ② : サーバーの準備
  • ③ : ドメイン設定
  • ④ : まとめ

html・cssファイルの準備

自分のパソコン上にhtmlとcssのファイルを準備します。htmlとcssとは何かというと、

html

htmlはサイトの構造を作るプログラミング言語です。位置などを調整することができます。

css

cssはサイトの見た目を作るプログラミング言語です。文字の色、大きさなどを調整できます。

テキストエディター

ファイルを準備する際にテキストエディターを準備する必要があります。これは何かというと、コードを書くのに使います。メモ帳でもokです。

僕は、vscodeを使っています。vscodeを使って、ファイルをデスクトップに準備してみましょう！

サーバーの準備

無料でサーバーを使うには、gihubというものを使います。これは海外のサーバーでファイルをアップロードするだけで、世界にwebサイトを公開することができます。

これが大変という方は、有料ですがとても使いやすいもののリンクを下記に貼っておきます。

おすすめサーバー2選

lolipop

lolipopは日本で今一番大きいサーバー会社です。キャンペーンを頻繁に行なっており、キャンペーンによっては価格がとても安くなったりします。今現在始めると、ドメインが永久無料になります。

mixhost

mixhostは、価格が安いです。個人的におすすめなサーバーです。

ドメイン設定

ドメインとは、インタネット上の住所のようなものです。主な役割としては、サイトの信頼性を上げることです。

ドメインといっても2種類あります。独自ドメインと無料ブログドメインがあります。

独自ドメイン

お名前.comなどで、ドメインを個人が契約するのが、独自ドメインというものです。

独自ドメインのメリットとしては、google検索で上位に表示されたり、アドセンスが通りやすくなったりします。

無料ブログドメイン

無料ブログドメインは、無料ブログサイト「はてな」や「アメーバ」などで無料でサイトを公開でき、そこで使われるurlのことです。

まとめ

公開する手順は、

ファイルを準備→サーバーにアップロード→ドメイン設定→公開される！

この流れとなります。
